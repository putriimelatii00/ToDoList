\# ToDoList



\## Deskripsi

ToDoList adalah aplikasi sederhana untuk mencatat dan mengelola daftar tugas. Aplikasi ini memiliki fitur login sederhana, CRUD data, dan pencarian data.



\## Cara Instalasi

1\. Clone repository.

2\. Masuk ke folder project.

3\. Buka project menggunakan editor.

4\. Jalankan aplikasi.



\## Cara Menjalankan

1\. Login menggunakan username dan password.

2\. Tambahkan data tugas.

3\. Edit atau hapus tugas.

4\. Cari tugas berdasarkan nama atau status.



\## Struktur Folder



ToDoList/

├── README.md

├── login.txt

├── data.txt

└── pencarian.txt



\## Riwayat Commit



\- Commit pertama membuat struktur project

\- Menambahkan fitur login

\- Menambahkan fitur CRUD

\- Menambahkan fitur pencarian

\- Merge seluruh fitur ke branch develop

\- Menyelesaikan merge conflict

\- Release versi 1.0
